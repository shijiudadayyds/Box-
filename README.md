🚀 Mihomo / Box 上帝模式配置

«🔥 智能分流 · 自动测速 · 流媒体优化 · OpenAI 优选
📦 适用于 Box 模块 / Mihomo (Meta) 内核»

---

✨ 项目特点

- ⚡ 自动测速选优节点
- 🔁 故障转移（节点挂了自动切换）
- 🧠 智能节点筛选（自动过滤垃圾节点）
- 🌍 多地区分组（港 / 日 / 新 / 美 / 韩 / 台）
- 🎬 流媒体优化（Netflix / Disney+ / TikTok）
- 🤖 OpenAI 智能优选（不锁节点，更稳定）
- 🌐 DNS 防污染 + 国内外分流
- 🎯 开箱即用，无需复杂配置

---

📦 使用方法

1️⃣ 获取配置

下载本仓库中的：

config.yaml

---

2️⃣ 填写订阅

打开 "config.yaml"，找到：

proxy-providers:

把：

url: "填写你的订阅链接"

替换成你的订阅地址。

---

3️⃣ 导入客户端

支持：

- 📱 Box（模块）
- 💻 Mihomo / Clash Meta
- 📦 其他兼容 Meta 内核客户端

---

4️⃣ 推荐默认选择（很重要）

导入后建议设置：

功能| 推荐选择
🌍 国外代理| 自动选择
🤖 OpenAI| OpenAI
💬 Telegram| 故障转移
🎬 Netflix| Netflix
🎥 Disney+| Disney+
🎵 TikTok| TikTok

---

🧠 工作原理（简单说明）

自动测速

自动检测所有节点延迟，选择最优线路。

智能筛选

自动排除以下节点：

- ❌ 回国节点
- ❌ 游戏节点
- ❌ 家宽节点
- ❌ 直连节点

---

OpenAI 优选逻辑

不会固定某一个节点，而是：

- 优先使用：
  - 🇯🇵 日本
  - 🇸🇬 新加坡
  - 🇺🇸 美国
- 且必须满足：
  - ✔ BGP / IEPL / IPLC / 流媒体线路

👉 避免 403 / 风控 / 不可用节点

---

流媒体优化

自动优选：

- Netflix
- Disney+
- TikTok

优先选择“可解锁节点”

---

⚠️ 常见问题

❓ 没有节点？

👉 检查是否填写订阅链接

---

❓ OpenAI 无法使用？

👉 尝试切换：

- OpenAI → 日本节点 / 美国节点

---

❓ Netflix 不解锁？

👉 切换到：

- Netflix → 日本 / 美国节点

---

❓ 延迟很高？

👉 尝试：

- 切换到「自动选择」
- 或手动选择地区节点
